Welcome to FN Portal's documentation!
=====================================


FN Portal (fn_portal) is a Django application that provides an
interface and API for fisheries assessment data collected using the
FN-II data model. It is built as an installable application that can
be added to other projects as needed.  It is currently a mimimum
viable product that provides views and templates for interactive
dispay of net sets and biological attributes of fish by species and
project.
